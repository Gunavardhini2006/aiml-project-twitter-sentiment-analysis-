# Twitter Sentiment Analysis using Machine Learning

## Project Overview

This project aims to classify Twitter posts (tweets) as Positive or Negative using Natural Language Processing (NLP) and Machine Learning techniques. The model is trained on the Sentiment140 dataset and uses TF-IDF Vectorization along with Logistic Regression for sentiment classification.

## Objective

To build a sentiment analysis model that can automatically identify the sentiment expressed in a tweet.

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Google Colab
* TF-IDF Vectorizer
* Logistic Regression

## Dataset

Dataset: Sentiment140

The dataset contains 1.6 million tweets labeled with sentiment values.

Sentiment Labels:

* 0 → Negative
* 4 → Positive

Dataset Source:
https://www.kaggle.com/datasets/kazanova/sentiment140

## Machine Learning Workflow

1. Load Dataset
2. Data Preprocessing
3. Feature Extraction using TF-IDF
4. Train-Test Split
5. Model Training using Logistic Regression
6. Model Evaluation
7. Sentiment Prediction on New Tweets

## Model Performance

Accuracy Achieved: 74%
